# Moonlight theme assets

Moonlight-only artwork lives in this folder so the daylight assets remain untouched.

- `backgrounds/`: moonlight parchment textures
- `ornaments/`: decorative corner, compass, and ship artwork
- `branding/`: moonlight title banner
- `equipment-icons/`: moonlight equipment-slot icons
- `spell-icons/`: moonlight spell-school and area icons

Add future moonlight-only assets here and update `css/theme-mode.css` or `js/theme-mode.js` rather than placing new `*-moonlight` files beside daylight assets.
